# Network Fundamentals

## What is Networking?

### Red
Una red es un conjunto de dispositivos conectados para intercambiar información.


## What is Internet?
Internet es una enorme red que está formada por muchas, muchísimas redes pequeñas dentro de ella.
Hay dos tipos de redes :
  Redes privadas : Es una red "local" de dispositivos que pertenecen a una organización, empresa o casa.
  Redes publicas : Es la red que conecta todas esas redes privadas.

## Indentifying Devices on a Network
IP addresses : En pocas palabras, una dirección IP (Internet Protocol Address) puede utilizarse como una forma de identificar un dispositivo (host) en una red durante un período de tiempo. Esa dirección IP puede luego ser asignada a otro dispositivo sin que la dirección en sí cambie.
Una dirección IP es un conjunto de números dividido en cuatro octetos. El valor de cada octeto forma parte de la dirección IP del dispositivo dentro de la red.

Por ejemplo:

192.168.1.10

Los cuatro octetos serían:

192 | 168 | 1 | 10

Este número se calcula mediante una técnica conocida como direccionamiento IP y subnetting (subredes)
IP pública : Una dirección IP pública se utiliza para identificar un dispositivo en Internet.
IP privada : Una dirección IP privada se utiliza para identificar un dispositivo dentro de una red privada, entre otros dispositivos de esa misma red.


Direcciones MAC : Los dispositivos de una red tienen una interfaz de red física, que es un pequeño circuito (chip) que se encuentra en la placa madre del dispositivo. A esta interfaz de red se le asigna en la fábrica una dirección única llamada dirección MAC (Media Access Control).
Una dirección MAC es un número hexadecimal de 12 caracteres dividido en pares y separado por dos puntos (:).

Por ejemplo:

a4:c3:f0:85:ac:2d

Los primeros 6 caracteres identifican a la empresa que fabricó la interfaz de red, y los últimos 6 son un identificador único para ese dispositivo.

Spoofing de MAC :Las direcciones MAC pueden ser falsificadas (spoofed).
A esto se le llama MAC Spoofing : Consiste en que un dispositivo se hace pasar por otro utilizando su dirección MAC.
